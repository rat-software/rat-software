Extension Installation Guide
  1. Download & Unzip:
    Download the rat-extension_v1.zip and extract (unzip) it to a folder on your computer.
  2. Open Extensions Menu:
    Open Google Chrome and type chrome://extensions in the address bar.
  3. Enable Developer Mode:
    Toggle the switch Developer mode in the top right corner of the page.
  4. Load Extension:
    Click the button Load unpacked that appeared in the top left.
  5. Select Folder:
    Select the folder you extracted in step 1. The extension should now appear in your list and is ready to use.
Note: Since this is a research tool, it is not listed in the Chrome Web Store and must be installed manually using these steps.
