Extension Installation Guide
  1. Download & Unzip:
    Download the rat-extension_v1.zip and extract (unzip) it.
  2. Open Extensions Menu:
    Open Google Chrome and type chrome://extensions in the address bar.
  3. Enable Developer Mode:
    Toggle the switch Developer mode in the top right.
  4. Load Extension:
    Click Load unpacked.
  5. Select Folder:
    Select the extracted folder.
