# RAT: Query Sampler

The Query Sampler is a backend module designed to automate keyword research and discovery. It interfaces directly with the **Google Ads API** to generate keyword ideas based on seed terms, regions, and languages defined within your RAT studies.

## 🚀 Features

* **Keyword Discovery**: Generates high-volume keyword ideas using the Google Ads Keyword Planner engine.
* **Localized Research**: Supports specific targeting by region (criterion ID) and language.
* **Search Metrics**: Retrieves average monthly search volumes and competition levels for each discovered idea.
* **Background Automation**: Uses a persistent scheduler to process pending keyword requests every 20 seconds.

---

## 🛠️ 1. Installation & API Setup

The Query Sampler requires a verified connection to the Google Ads API.

1.  **Install Dependencies**:
    Install all required packages for the controller and its sub-modules:
    ```bash
    python3 -m pip install --upgrade pip
    python3 -m pip install -r requirements_rat_query_sampler.txt
    ```
2.  **Google Ads Developer Token**: Obtain a developer token from the [Google Ads Center](https://developers.google.com/google-ads/api/docs/get-started/dev-token).
3.  **OAuth Credentials**: Create "Web Application" or "Desktop App" credentials in the [Google Cloud Console](https://console.cloud.google.com).
4.  **Generate Refresh Token**: Run the utility script to authorize your account and generate a refresh token:
    ```bash
    python3 generate_user_credentials.py --client_secrets_path=your_client_secret.json
    ```
5.  **Client Configuration**: Update your `google-ads.yaml` file (details below).

---

## 📄 2. Configuring `google-ads.yaml`

The Query Sampler expects this file to be located in the `query_sampler/` directory. Like your other config files, it is provided with **placeholders** that must be updated.

### Placeholder Reference:
| Key | Placeholder | Description |
| :--- | :--- | :--- |
| **developer_token** | `your_developer_token` | The token obtained from the Google Ads API Center. |
| **client_id** | `your_client_id` | OAuth2 client ID from Google Cloud Console. |
| **client_secret** | `your_client_secret` | OAuth2 client secret from Google Cloud Console. |
| **refresh_token** | `your_refresh_token` | The token generated by `generate_user_credentials.py`. |
| **login_customer_id** | `your_manager_id` | Optional: Your manager account ID if applicable. |

## 🖥️ 3. Management & Operation

### Individual Operation
If you are running the Query Sampler on a dedicated server:
* **Start**: `python3 query_sampler_controller_start.py`. This launches the background scheduler.
* **Stop**: `python3 query_sampler_controller_stop.py`. This script identifies and terminates all associated python3 processes using `psutil`.

---

## 🧹 4. Database Integration
The module uses a custom library (`db.py`) to manage its dedicated tables:
* **`qs_keyword`**: Holds seed keywords and their processing status.
* **`qs_keyword_idea`**: Stores the results and metrics returned from the API.
